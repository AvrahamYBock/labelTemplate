Printing Instructions

Important: Carefully read ALL the instructions before printing

  • Always run a test print first on plain sheet of paper. A blank paper template was included in purchase. This will ensure proper alignment of lables.

  • Check the following settings in Print Preview. When done click "Cancel", do not print from Print Preview.
    • Set margin to "None"
    • Set scale to 100%

  • Print from "System Dialog" by pressing Ctrl+Shift+P
    Set preferences as follows:
    • Media type: select "Label"/"Thicker Paper" or similar
    • Print Resolution/Quality: Best/Highest dpi

Press Ctrl+Shift+P to print. Click "Cancel" to cancel
