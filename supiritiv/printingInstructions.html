Printing Instructions

Important: Read the instructions first. (Following the instructions will bring you to your printer page, and you may not be able to read this.)

Always run a test print first on plain sheet of paper. A blank paper template was included in purchase. This will ensure proper alignment of labels.

  • Press Ctrl+Shift+P to open print dialog box
  • Select printer
  • Click on "Preferences"
  • Set preference for Media type to "Label","Thicker Paper" or similar
  • Set Print Resolution/Quality to Best/Highest dpi
  • Click OK then click Print
